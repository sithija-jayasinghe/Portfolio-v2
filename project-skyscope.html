The Problem

Most weather dashboards are static and boring. I wanted to build a dynamic dashboard that not only shows the current weather but also visualizes data trends like temperature changes and humidity levels over time, providing a deeper insight into weather patterns.

The Tech Stack

JavaScript
Chart.js
OpenWeather API
CSS Grid

Key Challenges

  • API Integration: Fetching and synchronizing data from multiple endpoints (Current Weather and 5-Day Forecast) to display a cohesive dashboard.
  • Data Visualization: Using Chart.js to render interactive graphs that update dynamically based on the selected city.
  • Error Handling: Managing API rate limits and handling cases where city names are invalid or data is unavailable.

The Result

A rich, interactive weather experience that goes beyond simple numbers.
